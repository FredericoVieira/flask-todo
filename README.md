# flask-todo

To do list using flask.

To set this project:

1. 	Environment:

	virtualenvwrapper must be installed.

	```bash
	mkvirtualenv flask-todo 
	...

	git clone https://github.com/FredericoVieira/flask-todo.git
	cd flask-todo
	pip install -r requirements.txt --upgrade
	```

2.	Running project:

	```bash
	python run.py
	```
